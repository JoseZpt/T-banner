# T-banner

Herramienta De Pesonalizacion Con Python2

# INSTALACIÓN:
```
$ apt update && apt upgrade 
$ pkg install git 
$ git clone https://github.com/Anonymous-Zpt/T-banner
$ cd T-banner 
$ chmod +x *
$ ./start
Tendrás Un Menú Y Hay Desides Tu. 
```
# Eliminación de banner
```
$ cd T-banner 

$ ./start -use --Delete
```
