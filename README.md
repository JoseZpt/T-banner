T-banner es una herramienta simple para personalizar la interfaz de termux

# Instalacion 
```
apt update && apt upgrade -y
pkg install git -y
git clone https://github.com/JoseZpt/T-banner
cd T-banner 
chmod +x *
```

# Menu
* Personalizar
`./start --banner`

* Eliminar banner
`./start --delete`

* Canal YouTube
`./start --josezpt`

* Mostrar banner
`./start --show`
